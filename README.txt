Cara Menjalankan

A. Menggunakan Terminal atau Command Prompt
1. Pastikan Anda memiliki Python terinstal di komputer Anda.
2. Simpan kode program di file bernama "kripto.py".
3. Buka terminal atau command prompt.
4. Navigasikan ke direktori tempat file "kripto.py" berada.
5. Jalankan program dengan perintah: python kripto.py
6. Antarmuka aplikasi akan muncul.
7. Pilih jenis cipher yang ingin digunakan (Vigenere, Playfair, atau Hill).
8. Pilih aksi yang ingin dilakukan (Enkripsi atau Dekripsi).
9. Masukkan kunci (pastikan panjang kunci minimal 12 karakter).
10. Anda dapat mengunggah file teks atau memasukkan teks secara langsung.
11. Klik tombol "Process" untuk mendapatkan hasil enkripsi atau dekripsi.

B. Menggunakan Visual Studio Code (VSCode)
1. Buka "Visual Studio Code".
2. Instal Python extension dari Microsoft (jika belum terinstal).
3. Buka folder tempat file "kripto.py" disimpan.
4. Klik kanan pada file "kripto.py" dan pilih "Run Python File in Terminal".
5. Aplikasi akan terbuka, dan Anda dapat mengikuti langkah-langkah seperti yang dijelaskan di atas.
